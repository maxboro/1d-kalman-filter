# Description
Simple example of Kalman filter in C++. Based on [this](https://www.kaggle.com/code/max398434434/1d-kalman-filter) notebook.

# How to run

#### Simulation:
```bash
make           # Builds the project
./main     # Runs the executable
```

#### Clean:
```bash
make clean     # Cleans up build files
```
